# Fonts

This directory contains font files for text rendering in the Rusteroids engine.

## Default Font

For development and testing, we recommend using open-source fonts:

- **DejaVu Sans** - Clear, readable sans-serif font (Public Domain)
- **Roboto** - Modern sans-serif from Google (Apache 2.0)
- **Liberation Sans** - Metric-compatible with Arial (SIL Open Font License)

## Installation

1. Download a font (e.g., DejaVu Sans from https://dejavu-fonts.github.io/)
2. Copy the `.ttf` file to this directory as `default.ttf`
3. Ensure the license is compatible with your project

## License Requirements

All fonts must have permissive licenses (MIT, Apache 2.0, OFL, Public Domain).
Include attribution in your game credits if required by the font license.
